# EasyColoc

**Simple expense sharing for roommates**

## What It Does

EasyColoc helps roommates share costs and track who owes what. Perfect for students and shared apartments.

### Main Features

- **Manage Housing** - Create rooms, add members, set roles
- **Track Expenses** - Add bills, split costs, see balances  
- **Smart Calculations** - Auto-calculate who pays whom
- **Invite Friends** - Send invites, manage access
- **User Roles** - Admin, room owner, member access
- **Trust System** - Build reputation with reliable payments

## Tech Stack

- **Framework**: Laravel 12.x
- **Database**: MySQL with migrations
- **Auth**: Laravel Breeze
- **Style**: Tailwind CSS

## Project Structure

```
app/
├── Controllers/     # Handle user actions
├── Models/          # Data models
└── Services/        # Business logic

resources/views/
├── layouts/         # Page templates
├── auth/            # Login/register
├── dashboard/       # Main app area
└── components/      # Reusable parts

database/
├── migrations/      # Database tables
└── seeders/         # Sample data
```
